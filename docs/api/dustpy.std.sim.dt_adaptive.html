dt_adaptive¶

dustpy.std.sim.dt_adaptive(sim)[source]¶

Function that returns the suggested adaptive timestep. By default DustPy uses adaptive integration schemes. The step size function is therefore simply returning the suggested step size.

Parameters

  • sim (Frame) – Parent simulation frame

  • Returns –

  • dt (float) – Time step
